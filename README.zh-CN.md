## жҰӮи§Ҳ

Confession жҳҜдёҖеҘ—еӣҙз»• VS Code е·ҘдҪңжөҒи®ҫи®Ўзҡ„йқҷжҖҒе®үе…ЁеҲҶжһҗе·Ҙе…·гҖӮе®ғжҠҠ AST и§„еҲҷеҲҶжһҗгҖҒLLM иҜӯд№үе®ЎжҹҘгҖҒWebview д»ӘиЎЁжқҝгҖҒHono API дёҺ CLI дёІжҲҗеҗҢдёҖжқЎй“ҫи·ҜпјҢе№¶жҠҠж•°жҚ®з»ҹдёҖдҝқеӯҳеңЁйЎ№зӣ®жң¬ең°зҡ„ `.confession/` еҘ‘зәҰдёӯгҖӮ

е®ғеқҡжҢҒдёүдёӘиҫ№з•ҢпјҡдёҚжү§иЎҢз”ЁжҲ·д»Јз ҒгҖҒдёҚдё»еҠЁе№Ійў„е·ҘдҪңеҢәгҖҒеҸӘжҸӯйңІйЈҺйҷ©дҝЎеҸ·дёҺдҝ®еӨҚе»әи®®гҖӮ

## дә®зӮ№

- **йқҷжҖҒдјҳе…Ҳ**пјҡеҸӘеҒҡ AST дёҺжЁЎеһӢеҲҶжһҗпјҢдёҚеҒҡ runtime жү§иЎҢгҖӮ
- **еҸҢеј•ж“Һжү«жҸҸ**пјҡй»ҳи®ӨдҪҝз”Ё `agentic`пјҢеӨұиҙҘж—¶еҗҢдёҖ task еҶ…иҮӘеҠЁеӣһйҖҖ `baseline`гҖӮ
- **VS Code еҺҹз”ҹдҪ“йӘҢ**пјҡDiagnosticsгҖҒHoverгҖҒCode ActionsгҖҒStatus Bar дёҺ Webview д»ӘиЎЁжқҝе…ұдә«еҗҢдёҖд»ҪзҠ¶жҖҒгҖӮ
- **дәӢд»¶й©ұеҠЁ AI е»әи®®**пјҡеҸӘжңүеңЁжү«жҸҸжҲ–е®Ўж ёдәӢд»¶д№ӢеҗҺпјҢе№¶йҖҡиҝҮй—Ёж§ӣгҖҒcooldown дёҺеҺ»йҮҚи§„еҲҷж—¶жүҚдјҡи§ҰеҸ‘гҖӮ
- **жң¬ең°ж–Үд»¶еӯҳеӮЁ**пјҡй…ҚзҪ®гҖҒжјҸжҙһгҖҒд»»еҠЎгҖҒе»әи®®еҝ«з…§дёҺеҲҶжһҗзј“еӯҳйғҪдҝқеӯҳеңЁ `.confession/*.json`гҖӮ
- **еҜјеҮәе®Ңж•ҙ**пјҡеҶ…е»ә `json`гҖҒ`csv`гҖҒ`markdown`гҖҒ`pdf`гҖҒ`sarif`гҖӮ

## йҮҚиҰҒжҸҗзӨә

> [!IMPORTANT]
> Confession дёҘж јйҷҗеҲ¶еңЁйқҷжҖҒеҲҶжһҗиҢғеӣҙеҶ…пјҢдёҚдјҡжү§иЎҢз”ЁжҲ·д»Јз ҒпјҢд№ҹдёҚдјҡеҗҜеҠЁзӣ®ж ҮжңҚеҠЎжҲ–еҒҡ runtime жіЁе…ҘгҖӮ

> [!TIP]
> ж—ҘеёёдҪҝз”Ёе»әи®®д»Ҙ `standard` дёәдё»пјҢ`quick` йҖӮеҗҲдҝқеӯҳеҗҺзҡ„еҚіж—¶еҸҚйҰҲпјҢ`deep` еҲҷдҝқз•ҷз»ҷеҸ‘зүҲеүҚжЈҖжҹҘгҖҒе®Ўи®ЎжҲ–е®Ңж•ҙе®үе…Ё reviewгҖӮ

> [!NOTE]
> й»ҳи®Өжү«жҸҸеј•ж“ҺжҳҜ `agentic`пјҢеҰӮжһңеӨұиҙҘпјҢеҗҺз«ҜдјҡеңЁеҗҢдёҖдёӘ task еҶ…иҮӘеҠЁеӣһйҖҖ `baseline`гҖӮ`pdf` еҜјеҮәе®һйҷ…иҝ”еӣһзҡ„жҳҜеҸҜжү“еҚ° HTMLпјҢйңҖиҰҒйҖҡиҝҮжөҸи§ҲеҷЁжҲ– Webview зҡ„жү“еҚ°жөҒзЁӢеҸҰеӯҳдёә PDFгҖӮ

## еҝ«йҖҹејҖе§Ӣ

### еүҚзҪ®иҰҒжұӮ

- Node.js `>= 18`
- pnpm `9.x`
- Go `1.21+`пјҢд»…еңЁйҮҚе»ә Go WASM analyzer ж—¶йңҖиҰҒ

### еҗҜеҠЁйЎ№зӣ®

```bash
pnpm install
pnpm dev
```

### LLM еҜҶй’Ҙ

еңЁ `web/.env.local` иҮіе°‘жҸҗдҫӣдёҖз»„ provider еҜҶй’Ҙпјҡ

```env
GEMINI_API_KEY="<set-in-web-env-local>"
NVIDIA_API_KEY="<set-in-web-env-local>"
```

### CLI зӨәдҫӢ

```bash
node confession-cli/bin/confession.js init

node confession-cli/bin/confession.js list --status open
node confession-cli/bin/confession.js status
```

## иҜӯиЁҖдёҺеҜјеҮә

- Webview ж”ҜжҢҒпјҡ`zh-TW`гҖҒ`zh-CN`гҖҒ`en`
- й»ҳи®ӨиҜӯиЁҖпјҡ`zh-TW`
- й…ҚзҪ®й”®пјҡ`confession.ui.language = auto | zh-TW | zh-CN | en`
- `auto` дјҡжҢҒз»ӯи·ҹйҡҸе®ҝдё»иҜӯиЁҖ
- `/api/export` ж”ҜжҢҒпјҡ`json`гҖҒ`csv`гҖҒ`markdown`гҖҒ`pdf`гҖҒ`sarif`
- жңӘжҢҮе®ҡ `locale` ж—¶пјҢеҗҺз«Ҝж №жҚ® `config.ui.language` и§ЈжһҗпјҢж— жі•еҲӨе®ҡж—¶еӣһйҖҖ `zh-TW`
